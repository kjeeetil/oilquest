OilQuest

This is a game that lets the player build an oil empire by acquiring acreage to do exploration, buying and selling of discoveries, do project development, sanction projects, execute projects, run oil production operations, and sell oil to refineries. The game is played on a map of the world, and the player can choose to play as a single player or as a multiplayer game. The game is turn-based, and the player can choose to do different actions in each turn. The game is played on a web browser, optimized for desktops.

The game is deployed to google cloud in a docker container.
